How can you protect yourself on the web

Here we have some general good pratices for you to ensure maximum security

Choose a secure password

A password is like a key to your house: If its bad, it is easier to break and go into your home and steal your stuff. In that case your private information like your online banking login data. Here you can check if your password is secure and read about some requirements for a good password. In general, passwords shouldnt contain dictionary words or concatenations of these words (for example, "House" is a bad password as well as "BigHouse")

use a password manager

You should have different passwords for different websites and change the frequently. To aid with this we recomend using a password manger program. Its like a safe which contains all of your password, locked awy by a strong and easy to remember master password.
You can read more about how to create a strong password (and remember it) here.

use transparent, open source services
