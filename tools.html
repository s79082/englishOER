Tips and Tools

Four our Tips and Tools we focused on the Firefox Webbrowser. If you're using Chrome or Safari you can lookup for extension which have the same features. The most common extensions their developers published them for all Browsers.

Ad-Blocker


FFrom Blocking Ads and Tracking Scripts we recommend the Browser Plugin "uBlock Origin". uBlock Origin is an efficient an easy to use FOSS (Free/libre Open Source Software) and you can add different blocking list for your special needs.  
uBlock Origin

Java-Script


With NoScript you can enable and disable Java-Script in your Browser. The standard configuration is suboptimal. For further reading you can read  this.
NoScript

Canvas Blocker


The extension CanvasBlocker can fake the hits for the Canvas-Api for prohibit fingerprinting. For the adjustment the "Stealth Settings" are recommend
CanvasBlocker

HTTPS


For using only encrypted websites HTTPSEverywhere care for you to use HTTPS Connections when its possible.
HTTPSEverywhere

Deleting tracking Parametrs in the URL


Neat URL delete known Tracking Parameters from Google, Microsoft, Facebook, YouTube and more from the URL
Neat-URL
